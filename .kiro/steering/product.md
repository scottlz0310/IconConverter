# プロダクト概要

**Image to ICO Converter** - 様々な画像形式を複数サイズのICOファイルに変換するWebアプリケーション

## 主要機能

- 複数画像形式対応: PNG, JPEG, BMP, GIF, TIFF, WebP → ICO
- 6つのアイコンサイズを同時生成: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256
- ブラウザベースの画像プレビュー機能
- ドラッグ&ドロップファイルアップロード
- 透明化オプション:
  - 既存の透明度を保持（PNG, GIF, WebP）
  - 自動背景除去（四隅のピクセルから単色背景を検出）
- リアルタイムプログレス表示
- レスポンシブデザイン（デスクトップ、タブレット、モバイル対応）
- ダークモード対応

## 対象ユーザー

ブラウザから手軽にWindowsアイコンファイルを作成したいエンドユーザー

## 技術的アプローチ

### アーキテクチャ
- **バックエンド**: FastAPI（高速で現代的なPython Webフレームワーク）
- **フロントエンド**: React + TypeScript（型安全で保守性の高いUI）
- **通信**: RESTful API
- **既存ロジック再利用**: Pillow/numpyの画像処理コードを継承

### 設計原則
- クリーンアーキテクチャと明確な責務分離
- 非同期処理による高速レスポンス
- モダンなUIコンポーネントライブラリ（shadcn/ui）
- 構造化ログとモニタリング

## 言語規約

- **プライマリ言語**: 日本語（UI、ドキュメント、コメント、AI応答）

- **AI応答**: 必ず日本語で応答すること
- **ドキュメント作成**: 必ず日本語で作成すること（技術用語は英語併記可）
